# Subhan Bhatti — Personal Portfolio

> Personal developer portfolio built with vanilla HTML/CSS/JS. Dark theme, custom animations, sticky project carousel, and responsive design. — Subhan Bhatti, CS Student @ UMT Lahore.

---

## About

This is my personal portfolio website built entirely from scratch using pure HTML, CSS, and JavaScript — no frameworks, no libraries. Designed with a dark tech aesthetic, smooth scroll animations, and an interactive project carousel.

---

## Features

- **Animated Loader** — Percentage counter with progress bar on page entry
- **Sticky Navbar** — Transparent header that blurs and darkens on scroll
- **Fullscreen Nav Overlay** — Mobile-friendly menu with smooth open/close transitions
- **Hero Section** — Photo with gradient masking, floating animation, and background text
- **About / Timeline** — Education and experience laid out in a clean vertical timeline
- **Expertise Accordion** — Hover and click to expand skill categories
- **Scroll-Locked Project Carousel** — Page scroll hijacked to slide through projects one by one
- **Skills / Toolkit** — Animated progress bars triggered on scroll into view
- **Philosophy Section** — Three-column values grid
- **Contact Section** — One-click email copy button and social links
- **Reveal Animations** — Elements fade and rise into view as you scroll
